# 参数重要性过估计实验预注册

## 研究问题

### H1：过估计是否存在

主统计量为同一检查点、同一采样条件下 `Naive - Oracle` 的配对差。

称为“显著且具有实际意义”必须同时满足：

1. 单侧检验 `E[Naive-Oracle] > 0` 经 Holm 校正后 `p < 0.05`；
2. 分层或聚类 bootstrap 的 95% 置信区间下界大于 0；
3. 平均偏差超过 Oracle 尺度的 10%。

### H2：两类修正是否有效

分别检验独立双采样与单采样协方差修正：

1. 误差对 ±5% Oracle 尺度通过 TOST 等价检验；
2. 相对 Naive 的绝对误差在配对单侧检验中显著降低。

不得用“未拒绝零偏差”代替等价性证明。

### H3：单采样是否优于双采样

报告三种预算口径：总样本数、梯度评估/反向传播次数、墙钟时间。

- `M=2`：Single-micro 与对称 matched Double 应逐次严格相等；
- `M>2`：若 `MSE(Single-micro)/MSE(Double-matched)` 的 95% bootstrap 上界小于 1，则支持单采样更优。

Matched Double 与 Single-micro 使用相同的 B 个样本和 M 个微批梯度；前者只平均 M/2 个不相交配对，后者平均全部交叉配对。

## 估计目标

局部一步 SGD 的节点目标为：

`omega* = gamma * E[u] * sum_q w_q E[v_q]`

同批次乘积包含 `Cov(u,v_q)/B`，因此单采样修正为：

`gamma * (mean(u) * mean(v_q) - s_uv,q / B)`

将批次分成 M 个等大微批后，对微批均值重复该推导并除以 M。该形式是二阶 U-statistic；无偏性不要求高斯分布，但要求相应采样独立性和有限二阶矩。

多节点路径积分必须估计更新梯度与每个节点梯度的交叉协方差。只在所有节点减去更新点方差不是一般无偏估计，因此仅作为 PPT 公式消融。

## 受控统计实验

- 分布：Gaussian、Student-t(5)、中心化对数正态、有限总体无放回；
- 解析问题：二次损失、线性回归、逻辑回归；
- `B={8,32,128}`，`M={2,4,8,16}`；
- 每格 20,000 次重复；
- 报告偏差、方差、MSE、解析偏差和 Gaussian 方差公式。

有限总体无放回作为理论边界测试，不并入 IID 主结论。

## 真实模型定点实验

- MNIST MLP；
- CIFAR-10 ReLU+MaxPool SimpleCNN；
- CIFAR-10 Softplus+AvgPool SmoothCNN；
- 固定类别平衡总体，最大 10,000 个样本；
- 初始化、25%、50%、100% 训练检查点；
- 主设计 `B={16,64,256}`、`M={2,4,8}`、5 个训练种子；
- 本机代表性 CIFAR 配置可减少总体、批量网格与重复数，但不得改变模型平滑性对照。

指标按模型、层和可选逐参数记录：有符号偏差、绝对偏差、相对偏差、MSE、Spearman 相关、重要性总和、守恒误差、梯度评估数、耗时和峰值显存。

## 积分参考认证

任何固定阶高斯求积都不定义为数学真值。数值参考同时使用：

1. 全数据自适应 Gauss-Kronrod 向量积分；
2. 独立复合梯形网格加密；
3. 最细网格与前一级网格的相对 L2 差；
4. `sum(importance)=L(theta0)-L(theta1)` 端点损失守恒。

只有自适应算法成功、独立方法一致、网格收敛且两种结果均满足守恒容差时，才标记为已认证。该认证是数值证据，不是逐参数严格解析真值证明。

## 持续学习

完整目标设计：

- Permuted-MNIST：5 任务、MLP、每任务 5 epochs；
- Split-CIFAR-100：10×10 类、ResNet-18、多头 Task-IL、每任务 30 epochs；
- 方法：Fine-tuning、EWC、Naive、Double、Single-direct、Single-micro M=4/8；
- 调参种子 100–102，最终种子 0–4，正则强度 `10^-3...10^3`。

本机资源受限运行必须保持任务数不变，并在报告中明确数据子集、epoch、强度网格和种子数，不得与完整矩阵混称。

报告最终平均准确率、平均遗忘、后向迁移、重要性 L1、训练时间、样本数和峰值显存。

## 统计与多重比较

- 三个主假设使用 Holm 校正；
- 层级/参数级探索结果使用 BH-FDR；
- 置信区间使用 10,000 次按条件分层的 Monte Carlo 分块 bootstrap，或按训练种子聚类 bootstrap；
- 不同数据集、优化器或采样协议结论不一致时，按适用条件分别报告。

## 适用边界

严格无偏主张限定于普通无动量、无权重衰减 SGD、明确的 IID 采样协议和有限二阶矩。以下条件单独作为压力测试：无放回抽样、数据增强、大步长、Momentum、AdamW、ReLU/MaxPool 非光滑路径。
